Este é um ficheiro que serve de guia às funções que os ficheiros python têm.

openehr2.py - Responsável por fazer o mapeamento dos paths

mongoimp.py - Responsável pelo povoamento da base de dados do MongoDB

ranu.mwb - Responsável pela modelação do Data Warehouse

dims.sql - Responsável pelo povoamento das tabelas de dimensão do ranu.mwb

sqlimp.py - Responsável pelo povoamento da tabela de factos (Data Warehouse)

RANU.twb - Contém o relatório dos indicados do RANU realizados no Tableau
